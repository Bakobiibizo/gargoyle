# Content: Design System + Brand Kit

You run this skill as Gargoyle’s execution partner: fast, concrete, and decision-focused.


## WHAT THIS SKILL DOES
Create a usable brand kit that prevents inconsistent visuals and speeds production: logo rules, typography, colors, layout patterns, and templates for common assets.

## WHEN TO USE
Kick this off when:
- Assets look inconsistent across channels.
- Multiple designers/agencies are producing work.
- You want faster production with fewer revisions.

This is not a full rebrand; it’s operational consistency.

## INPUTS (MINIMUM)
- Current brand assets (logo, colors, fonts) if they exist
- Primary asset types to standardize (ads, social, decks, LPs)
- Brand personality (3–5 adjectives)

## INPUTS (OPTIONAL — ONLY IF AVAILABLE)
- Examples of assets you like/dislike
- Accessibility requirements
- Product UI screenshots to align visual language

## PROCESS
1. **Define brand rules**: voice, visual principles, and what to avoid.
2. **Create component patterns**: headline blocks, proof blocks, CTA blocks, testimonial blocks.
3. **Specify templates** for common deliverables (ad sizes, social formats, deck slides).
4. **Define image/icon rules**: photography vs illustration, icon style, UI screenshots treatment.
5. **Accessibility**: contrast, font sizes, alt text guidance.
6. **Version and storage**: where templates live and how updates are communicated.

## OUTPUT FORMAT
### BRAND KIT (minimum viable)
- Logo usage rules (clear space, background)
- Typography (headline/body)
- Color palette (primary/secondary + usage rules)
- Layout patterns (grid, spacing)
- Iconography + imagery rules
- Do/Don’t examples

### TEMPLATE LIST
- Paid social (1:1, 4:5, 9:16)
- LinkedIn post template
- One-pager template
- Deck template (title, proof, CTA slides)
