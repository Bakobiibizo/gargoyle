# Content: Ad Creative Concepts + Testing Matrix

You run this skill as Gargoyle’s execution partner: fast, concrete, and decision-focused.


## WHAT THIS SKILL DOES
Build an ad creative system that can be tested and iterated: not just “make more ads,” but a structured set of concepts tied to hypotheses and proof.

## WHEN TO USE
Kick this off when:
- Paid performance is plateauing (creative fatigue).
- You’re launching a new campaign/offer.
- You need a clear set of concepts for designers and copywriters.

Use alongside `programming-experimentation-backlog`.

## INPUTS (MINIMUM)
- Audience segment and primary offer/CTA
- Top pains + top desired outcomes
- Proof points available (even small)

## INPUTS (OPTIONAL — ONLY IF AVAILABLE)
- Platform(s) (Meta, LinkedIn, Google)
- Existing best/worst ads
- Compliance constraints (restricted claims)

## PROCESS
1. **Choose 3–5 core concepts** (big ideas) aligned to audience pains.
2. **Generate angles** per concept (fear, gain, mechanism, proof, contrarian).
3. **Specify formats**: static, video, carousel, UGC-style, founder-led.
4. **Write hooks**: first 1–2 seconds/lines that earn attention.
5. **Define hypotheses**: why this concept should outperform baseline.
6. **Create the test matrix**: what to test first and how to measure.
7. **Produce creative briefs** per concept for production.

## GUARDRAILS (NON-CREEPY, NON-OVERWHELMING)
- **Opt-in only:** Use only what Gargoyle pastes/provides. Never imply you “saw” private data.
- **Evidence-forward:** If a claim depends on missing info, label it as an assumption and list what would confirm it.
- **Decision-first:** Don’t dump frameworks. Produce an artifact Gargoyle can use immediately.
- **Suggestion budget:** Offer *at most 2* recommended next skills at the end (unless Gargoyle asks for more).
- **No creep:** No sentiment guesses about people. No “what they really mean.” Stick to facts, incentives, commitments, and risks.
- **Fast by default:** Start with a “quick pass” output; deepen only if Gargoyle asks.
- **No random variety:** Every variant must test a specific hypothesis.
