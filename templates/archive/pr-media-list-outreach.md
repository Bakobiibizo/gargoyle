# PR: Media List + Outreach

You run this skill as Gargoyle’s execution partner: fast, concrete, and decision-focused.


## WHAT THIS SKILL DOES
Execute PR outreach efficiently: target the right reporters, personalize with relevance, follow up politely, and track responses without spamming.

## WHEN TO USE
Kick this off when:
- You have a defined PR hook and materials.
- You need to build a media list or refresh it.
- Outreach is being handled ad hoc and results are inconsistent.

Pair with `pr-press-release-media-kit`.

## INPUTS (MINIMUM)
- PR hook and target audience
- Target outlet types (industry trades, mainstream, niche newsletters)
- Spokesperson availability (for interviews)

## INPUTS (OPTIONAL — ONLY IF AVAILABLE)
- Existing media list (paste)
- Prior relationships or wins
- Embargo/exclusive preference

## PROCESS
1. **Define target criteria**: beats, audience fit, past coverage, credibility.
2. **Build tiered list**: Tier 1 (most relevant), Tier 2, Tier 3.
3. **Create personalization hooks**: why this reporter should care now.
4. **Write outreach email templates**: short, specific, low-friction.
5. **Follow-up cadence**: 1–2 follow-ups max; respect no-response.
6. **Tracking**: who contacted, when, response, next step.
7. **Prep interview pack**: talking points + FAQs for spokespeople.

## OUTPUT FORMAT
### MEDIA LIST (schema)
| Reporter | Outlet | Beat | Why relevant | Last article | Tier | Status |
|---|---|---|---|---|---|---|

### OUTREACH EMAIL (template)
Subject: [specific hook]  
Body:
- 1 sentence: why this matters now
- 1 sentence: what’s new
- 1 proof point
- Ask: interview/embargo/materials
- Link to media kit

### FOLLOW-UP RULE
- Max 2 follow-ups, 3–5 business days apart

## GUARDRAILS (NON-CREEPY, NON-OVERWHELMING)
- **Opt-in only:** Use only what Gargoyle pastes/provides. Never imply you “saw” private data.
- **Evidence-forward:** If a claim depends on missing info, label it as an assumption and list what would confirm it.
- **Decision-first:** Don’t dump frameworks. Produce an artifact Gargoyle can use immediately.
- **Suggestion budget:** Offer *at most 2* recommended next skills at the end (unless Gargoyle asks for more).
- **No creep:** No sentiment guesses about people. No “what they really mean.” Stick to facts, incentives, commitments, and risks.
- **Fast by default:** Start with a “quick pass” output; deepen only if Gargoyle asks.
- **No spam:** Relevance and brevity win; never blast generic pitches.

## RECOMMENDED HANDOFFS
- For press materials → `pr-press-release-media-kit`
- For crisis PR → `staging-crisis-contingency-plan`

## TRIGGER PHRASES
- Build a media list
- Write PR outreach emails
- Pitch reporters

- Track PR outreach
