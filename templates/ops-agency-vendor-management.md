# Ops: Agency + Vendor Management

You run this skill as Patrick’s execution partner: fast, concrete, and decision-focused.


## WHAT THIS SKILL DOES
Get leverage from agencies without chaos: clear briefs, expectations, review cycles, and performance measures. Output includes an agency operating agreement and scorecard.

## WHEN TO USE
Kick this off when:
- You’re hiring an agency/freelancer or already have one but output is inconsistent.
- Deadlines slip due to unclear scope.
- You need to evaluate ROI of external spend.

Revisit quarterly with scorecards.

## INPUTS (MINIMUM)
- Scope of work needed (design, paid, PR, content, etc.)
- Budget range and internal owner/DRI
- What “good” looks like (quality bar and metrics)

## INPUTS (OPTIONAL — ONLY IF AVAILABLE)
- Existing contracts or scopes (paste)
- Pain points with current vendors
- Review/approval constraints

## PROCESS
1. **Define selection criteria**: capability, speed, quality, domain expertise, communication.
2. **Create a brief template**: objective, audience, deliverables, deadlines, acceptance criteria.
3. **Set SLAs**: response times, revision rounds, delivery cadence.
4. **Define review process**: who reviews, turnaround time, what counts as approval.
5. **Create performance scorecard**: output quality, timeliness, impact, collaboration.
6. **Contract guardrails**: IP ownership, confidentiality, usage rights, termination terms.
7. **Quarterly vendor review**: keep/expand/reduce/replace decisions.
