# Content: Copywriting (Shortform)

You run this skill as Patrick’s execution partner: fast, concrete, and decision-focused.


## WHAT THIS SKILL DOES
Create shortform copy that gets attention and drives action without sounding like marketing spam. Includes hooks, variants, and platform-specific constraints.

## WHEN TO USE
Kick this off when:
- Writing emails, ads, social posts, release notes, or announcements.
- You need multiple variants for testing.
- You want to adapt messaging blocks to a channel quickly.

## INPUTS (MINIMUM)
- Channel + format (email, LinkedIn post, ad, etc.)
- Audience segment
- Single CTA

## INPUTS (OPTIONAL — ONLY IF AVAILABLE)
- Messaging blocks or proof points
- Tone constraints (formal, direct, playful)
- Length constraints (characters/words)

## PROCESS
1. **Choose the hook type**: pain, contrarian, proof, question, mechanism.
2. **Draft 3–5 variants** with different angles but same message spine.
3. **Add proof**: one concrete detail that signals credibility.
4. **Trim**: remove adjectives, keep verbs and facts.
5. **Compliance/brand check**: avoid banned claims or risky language.
6. **Finalize**: choose best variant + create 2 backups for testing.

## OUTPUT FORMAT
### OUTPUTS (choose format)
**Email**
- Subject options (5)
- Preheader options (3)
- Body (short)
- CTA

**Ad**
- Primary text variants (5)
- Headline variants (5)
- Description variants (3)

**Social**
- Post variants (3–5)
- Hook options (5)
- CTA options (5)
